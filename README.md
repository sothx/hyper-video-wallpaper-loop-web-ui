# Hyper OS ZRAM Writeback For Web UI

## 模块简介

该Magisk模块可以可视化管理并扩展增强有关Hyper OS动态壁纸的功能

目前支持以下功能：

- 快捷访问 AI 动态壁纸

- 设置动态壁纸循环播放

## Web UI 开发模式和生产模式

开发阶段：

修改webroot下的web.config.js，将env修改为dev

生产阶段:

修改webroot下的web.config.js，将env修改为prod
